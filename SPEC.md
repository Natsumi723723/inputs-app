# INPUTS — アプリ仕様書

> 個人の日常インプットを記録・振り返るための、シングルファイルHTMLアプリ。

---

## コンセプト

「思考の足跡を残す」

読んだ・観た・感じた・思いついた——あらゆるインプットを気軽に記録し、後から感情タグやメディアで振り返る。記録ツールではなく「収集」の感覚。

---

## 技術構成

| 項目 | 内容 |
|------|------|
| 実装 | シングルファイル HTML / CSS / JS（`app.html`） |
| フレームワーク | なし（バニラJS） |
| データ永続化 | `localStorage`（キー: `inputs_v1`） |
| フォント | M PLUS Rounded 1c（Google Fonts） |
| サーバー | `npx serve` でローカル起動（ポート 3333） |

---

## データモデル

### Entry（1件のインプット）

```ts
{
  id:        string,        // uid() — Date.now(36) + random
  type:      CatId,         // メディアカテゴリID
  title:     string,        // タイトル（必須）
  url:       string,        // URL（任意）
  note:      string,        // メモ（任意）
  createdAt: ISO8601,       // 記録日時
  tokimeki:  1 | 2 | 3 | 4 | 5,  // ときめき度
  emotions:  EmoId[],       // 感情タグ（複数可）
}
```

## 画面構成

### 共通ヘッダー

- **ロゴ（INPUTS）** — タップでタイムラインへ
- **テーマドット** — 4色のカラーピッカー
- **検索ボタン（🔍）** — タップで検索バー開閉。入力中はピンク背景でアクティブ表示
- **検索バー** — タイトル・メモ・URL横断。入力があると ✕ クリアボタンが表示される

### タブ（3種）

| タブ | アイコン | 内容 |
|------|----------|------|
| タイムライン | 📋 | 時系列リスト + フィルター |
| 本棚 | 📚 | 本のビジュアル表現 |
| 分析 | 📊 | 統計・集計ダッシュボード |

---

## タイムライン画面

- エントリを日付ごとにグループ化して表示
- 日付ヘッダーはアクセントカラー + 左ボーダーで視認性向上

### フィルター

2ボタン方式（ドロップダウン付き）：

1. **🌐 メディア** — ALL または8種のカテゴリから選択
2. **💖 感情** — ALL または6種の感情タグから選択

フィルター適用順：`検索キーワード → 感情タグ → メディアカテゴリ`

メディアの件数はキーワード＋感情でフィルター後の値を表示（選択時の予測件数）。

### エントリカード

- カテゴリアイコン（カラー背景）
- タイトル
- メモの冒頭
- URL（ドメイン名）
- 記録時刻
- ときめき度ドット（♥ で 1〜5 段階）

---

## 本棚画面

インプットを「本の背表紙」としてビジュアル表現。

### 本の見た目

- **幅** — ときめき度が高いほど太い（+8px/段階）、メモが長いほどわずかに太い
- **高さ** — ときめき度が高いほど高い（+9px/段階）
- **色** — メディアカテゴリ別のグラデーション
- **本文（縦書き）** — タイトルをバーティカルテキストで表示
- **疑似乱数** — `hsh()` 関数でIDをシードにしてサイズにばらつきを付与（同じエントリは常に同じサイズ）

### 表示モード切り替え（トグル）

| モード | 内容 |
|--------|------|
| 日 | 日付ごとにグループ化 |
| 月 | 月ごとにグループ化 |
| カテゴリ | メディアカテゴリごとにグループ化 |

選択中のタブはアクセントカラーで強調。

### 検索

ヘッダーの検索ボタンで開閉（タイムラインと共有）。タイトル・メモ・URL横断検索。

---

## 分析画面（ダッシュボード）

### 統計カード（4枚）

- 今日の件数
- 今月の件数
- 今年の件数
- 累計件数

### メディア別グラフ

- カテゴリごとの件数をプログレスバーで表示
- 最大値を100%として相対表示

### 感情タグ別グラフ

- タグごとの件数 + 全体に対する割合（%）
- 各タグのテーマカラーで着色

## 追加 / 編集シート

FABボタン（+）またはエントリの編集アイコンから開く。

### 入力フィールド

| フィールド | 必須 | 備考 |
|-----------|------|------|
| タイトル | ✅ | 未入力時は赤ハイライト + フォーカス |
| カテゴリ | ✅ | 8種のチップから選択 |
| URL | — | 折りたたみ式。入力するとドメイン名をプレビュー、タイトル自動補完 |
| メモ | — | フリーテキスト |
| 感情タグ | — | 複数選択可。選択数に応じてときめき度を自動更新 |
| ときめき度 | ✅ | ♥ ボタンで1〜5選択（感情タグから自動セットも可能） |

### 編集モード

詳細ポップアップの鉛筆アイコン → 追加シートが「編集する」に切り替わり、既存データを復元。「更新する」で上書き保存。

---

## 詳細ポップアップ

エントリタップで表示。

- **ヘッダー** — タイトル全文（可変長）、鉛筆アイコン（編集）、✕（閉じる）
- **バッジ** — カテゴリアイコン＋名前
- **ときめき度** — ♥ 5段階表示 + ラベル
- **感情タグ** — タグがある場合のみ表示
- **メモ**
- **URL** — 外部リンク（別タブ）
- **記録日時**
- **削除ボタン** — 左下。確認ダイアログ付き
